# 🪙 Coin Flip Simulator

A simple JavaScript mini project that simulates flipping a coin and displays either **Heads** or **Tails** at random.

Built as part of my **JavaScript Refresher** series to practice DOM manipulation, event handling, conditional logic, and random number generation.

---

## 🚀 Features

- Flip a virtual coin with one click
- Random Heads or Tails result
- Dynamic result display
- Visual coin icon updates
- Instant user feedback
- Clean, centered card UI

---

## 🧠 What I Practiced

- Selecting DOM elements with `getElementById()`
- Handling button click events with `addEventListener()`
- Generating random values with `Math.random()`
- Using conditional statements (`if / else`)
- Updating page content dynamically with `textContent`
- Building a simple interactive application
- Connecting HTML, CSS, and JavaScript together

## 🎯 How It Works

1. The user clicks the **Flip Coin** button.
2. JavaScript generates a random number using `Math.random()`.
3. If the number is less than `0.5`, the result is **Heads**.
4. Otherwise, the result is **Tails**.
5. The page updates instantly without refreshing.
