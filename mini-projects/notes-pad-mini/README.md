# 📝 Notes Pad Mini

A simple JavaScript mini project that allows users to write, save, and clear notes.  
Built as part of my **JavaScript Refresher** series to practice DOM manipulation, localStorage, and event handling.

---

## 🚀 Features

- Write and save notes in the browser
- Notes persist after page refresh using `localStorage`
- Clear notes with a single click
- Input validation to prevent empty saves
- Status feedback messages (saved, loaded, cleared)
- Clean and responsive dark UI
- Smooth button interactions and focus states

---

## 🧠 What I Practiced

- Selecting DOM elements with `getElementById`
- Handling user interactions with `addEventListener`
- Creating and organizing functions for clean logic
- Using `localStorage` to store and retrieve data
- Working with `DOMContentLoaded` to load saved data
- Input validation and trimming user input
- Updating the UI dynamically (status messages)
- Improving UX with feedback and interaction states

## 💡 How It Works

1. Type your note in the textarea
2. Click **Save Note** to store it in the browser
3. Refresh the page → your note is still there
4. Click **Clear Note** to remove the saved note
