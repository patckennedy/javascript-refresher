# 🔁 Palindrome Checker

A simple JavaScript mini project that checks whether a word or phrase is a **palindrome** (reads the same forward and backward).

Built as part of my **JavaScript Refresher** series to practice string manipulation, conditional logic, and DOM interaction.

---

## 🚀 Features

- User input field for entering text
- Button to check if the input is a palindrome
- Ignores:
    - Spaces
    - Special characters
    - Letter casing (case-insensitive)
- Real-time result feedback:
    - ✅ Palindrome
    - ❌ Not a palindrome
- Input validation (handles empty input)
- Clean and centered UI

---

## 🧠 What I Practiced

- Handling user input with `input.value`
- Using `addEventListener` for button clicks
- String manipulation:
    - `.toLowerCase()`
    - `.replace()` (cleaning input)
    - `.split().reverse().join()`
- Conditional logic (`if/else`)
- Updating the DOM dynamically (`textContent`)
- Basic input validation
