# ⏱️ Mini Stopwatch

A simple JavaScript mini project that functions as a stopwatch with start, stop, and reset controls.  
Built as part of my **JavaScript Refresher** series to practice DOM manipulation, timing functions, and event handling.

---

## 🚀 Features

- Start, stop, and reset the stopwatch
- Real-time time tracking (minutes, seconds, milliseconds)
- Accurate timing using `Date.now()`
- Clean and responsive UI
- Interactive buttons with hover and active states
- Disabled button states for better user experience

---

## 🧠 What I Practiced

- Selecting DOM elements with `getElementById`
- Handling user interactions with `addEventListener`
- Using `setInterval()` for real-time updates
- Working with timestamps using `Date.now()`
- Formatting time (mm:ss.ms)
- Managing application state (start, stop, reset logic)
- Improving UX with button states and feedback

---

mini-stopwatch/

---

## 💡 How It Works

1. Click **Start** to begin the stopwatch  
2. Click **Stop** to pause the time  
3. Click **Reset** to clear the timer back to zero
