# 🎲 Random Quote Generator (Mini Project)

A simple and fun JavaScript project that displays a new motivational quote every time the user clicks the **Generate New Quote** button.  
This mini project is part of my **JavaScript Refresher** series.

---

## 🚀 Features

- Generate a new random quote on button click
- Quotes stored in a local array (no API required)
- Prevents showing the same quote twice in a row
- Displays a quote automatically on page load
- Button interaction feedback (click animation + disabled state)
- Clean, modern UI with smooth styling and transitions

---

## 🧠 What I Practiced

- Selecting DOM elements with `getElementById()`
- Handling user events with `addEventListener()`
- Generating random values using `Math.random()`
- Working with arrays of objects
- Updating UI dynamically with `textContent`
- Basic state management (`lastIndex` to avoid repeats)
- Improving UX with button state control
