# 🔢 Even-Odd Numbers Summarizer

An interactive JavaScript mini project that allows users to input numbers and automatically separates them into **even** and **odd** categories.

This project reinforces **DOM manipulation, event handling, arrays, and conditional logic**.

---

## 🎯 Goal

- Accept user input (numbers)
- Process and categorize numbers into:
  - Even numbers
  - Odd numbers
- Display results dynamically in the UI

---

## 🚀 Features

- Input numbers separated by commas (e.g. `1,2,3,4`)
- Also supports continuous digits (e.g. `1234`)
- Automatically filters invalid input
- Displays results instantly on button click
- Clean and simple UI

---

## 🧠 Concepts Used

- DOM selection (`getElementById`)
- Event listeners (`click`)
- Arrays and loops (`for...of`)
- Conditionals (`if`)
- Modulus operator (`%`) for even/odd logic
- String methods (`split`, `trim`)
- Array methods (`map`, `filter`, `push`)
- Dynamic UI updates (`textContent`)

---

## ⚙️ How It Works

1. User enters numbers in the input field
2. Clicks the **Analyze** button
3. JavaScript:
   - Parses the input into an array
   - Separates even and odd numbers
   - Updates the UI with results
