# 🎨 RGB Color Mixer (Mini Project)

A simple interactive JavaScript project that allows users to mix colors using RGB sliders and instantly preview the result in real time.  
This mini project is part of my **JavaScript Refresher** series.

---

## 🚀 Features

- Adjust Red, Green, and Blue color values using sliders
- Live color preview updates instantly
- Displays both RGB and HEX color values
- Smooth UI transitions for color updates
- Responsive centered layout
- Clean dark-themed modern interface

---

## 🧠 What I Practiced

- Selecting DOM elements with `getElementById()`
- Handling slider input events with `addEventListener()`
- Updating UI dynamically with JavaScript
- Working with functions and reusable logic
- Using template literals for dynamic strings
- Converting RGB values to HEX format
- Looping through arrays with `forEach()`
- Manipulating inline styles with JavaScript
- Structuring beginner-friendly responsive layouts with Flexbox

---

## 📂 File Structure

```bash
rgb-color-mixer-mini/
│
├─ index.html   # Page structure and slider controls
├─ style.css    # Layout, colors, spacing, and transitions
└─ script.js    # RGB logic, HEX conversion, and live updates
```

---

## 💡 How to Use

1. Open **index.html** in your browser
2. Move the RGB sliders
3. Watch the preview color update instantly
4. View both RGB and HEX color codes live
